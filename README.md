# vuejemplo
Ejemplo de vue usando Api con mongoose

1- Descargamos el binario en: https://mongoose.ws/binary/
2- Colocamos el exe justo en la raiz del proyecto(al mismo nivel que index.html) y corremos el programa
3- colocamos en el navegador: http://localhost:8000/index.html (el puerto lo indica el ejecutable puede cambiar )
